# [二叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree/)

```js
var maxDepth = function (root) {
    if (root === null) {
        // 查询到了底部
        return 0
    }

    const left = maxDepth(root.left)
    const right = maxDepth(root.right)

    return Math.max(left, right) + 1
};
```
